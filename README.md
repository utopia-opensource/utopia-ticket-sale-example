# utopia-ticket-sale-example
A small example of creating a website selling tickets for an event for Utopia Crypton

## Installation

```bash
composer update
cp example.env .env
cd public_html
cp example.htaccess .htaccess
```

* make changes to `.env`;
* import db tables dump from `resources/tables.sql`;
